Question-03:What is even delegation in the DOM?

Event delegation is a technique in JavaScript where instead of attaching event listeners to individual elements, you attach a single event listener to a parent element and take advantage of event bubbling to handle events for its child elements.

Question-01:What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event triggered on an element is passed (bubbled) up to its parent elements in the DOM tree, until it reaches the root of the document . This allows parent elements to listen for events that occur on their child elements.
